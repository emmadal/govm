govm is a tool for managing multiple Go versions on macOS, Linux

## Example

```sh
$ govm install 1.24.1
  Switched to go1.24.1. Run 'source ~/.zshrc' or Run 'source ~/.bashrc' and restart your terminal to apply permanently.

$ govm use 1.24.1
  Switched to go1.24.1. Run 'source ~/.zshrc' or Run 'source ~/.bashrc' and restart your terminal to apply permanently.
```
